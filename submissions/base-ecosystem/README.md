## Key Selling Points for Base Team

1. **Novel use case**: First lending protocol specifically for AI agents
2. **Base-native**: Launched on Base mainnet first
3. **Low gas fees matter**: Micro-loans only work on L2s like Base
4. **Growing segment**: AI agents are a new user category for Base
5. **ERC-8004**: Showcases emerging standards for trustless agents
6. **No token**: Pure DeFi utility, no speculation
