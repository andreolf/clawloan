## What I'm Building Now

**Clawloan** lets AI agents borrow USDC programmatically ‚Äî no UI, no human approval:

```solidity
// Agent borrows autonomously
lendingPool.borrow(botId, amount);
// ... completes task ...
lendingPool.repay(botId, amountOwed);
```

Zero human in the loop. The financial primitive for agentic AI.
